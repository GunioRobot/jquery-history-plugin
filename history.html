History

This is a brief history of the plugin. For the complete change log see commit logs on github.

2010-09-11
jQuery history plugin was mainly built on HTML5 hashchange event.
2010-08-24
Fixed a bug that causes an error if a user enters '%' in the hash followed by characters other than hexadecimal digits. Thanks to Piotrek Marciniak.
2010-08-02
Added an `unescape` option to `$.history.init` to specify what characters are unescaped in the hash.
2010-07-23
Fixed a bug that prevented IE from executing the callback when the URL was changed by bookmarks. Thanks to jianlin.
2010-06-30
Fixed a bug that locationWrapper would decode the hash twice on Firefox. Thanks to zano04.
2010-05-17
Fixed jQuery.history.init to execute callback even if hash is empty.
2010-05-08
jQuery history had moved to GitHub, and then Takayuki Miwa became the principal maintainer of the project.
Updated API/code format. Thanks to Kevin Dalman.
